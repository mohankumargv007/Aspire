# Aspire

### Clone this repo
    Checkout to master branch

### Create Database
    create database Aspire

### Give Proper Credentials In .env
    Give Database and SMTP credentails in .env

### Run Composer Update
    composer update

### Run migrations
    php artisan migrate

### Run db seed
